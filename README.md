# Beacon

Beacon 是一个轻量级的消息传输协议，专为物联网和实时通信场景设计。

## 功能特性

- **发布/订阅模式**：支持主题订阅和消息发布
- **请求/响应模式**：支持客户端间的请求-响应通信

## 使用方法

导入包：

```go
import "github.com/snple/beacon/client"
import "github.com/snple/beacon/core"
```

更多使用示例请参考 `examples/` 目录。

## Beacon 与 MQTT 5.0 对比

| 对比项 | Beacon | MQTT 5.0 |
| --- | --- | --- |
| 通信模型 | 发布/订阅 + 请求/响应 | 发布/订阅 |
| 主题与路由 | 主题订阅，支持请求/响应场景 | 主题订阅，需额外约定实现请求/响应 |
| 设计目标 | 轻量、面向物联网与实时通信 | 轻量、面向物联网消息传输 |
| 协议栈依赖 | 以库形式集成应用 | 独立协议，需要 Broker |
